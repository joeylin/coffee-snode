#简介
这是一个nodejs的博客

采用 `Express.js 3.0` mvc框架，`MySQL`数据库和`Jade`模版引擎

使用CoffeeScript编写

#build and run
将CoffeeScript编译成js

打开命令行build: 
```
build.bat
```

打开另一个run:

```
strat.bat
```

请删除掉views/common/_layout.jade中的百度统计代码！
